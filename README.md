## What We Do

We are building a **bottom-up, fully open-source robotics platform**.

From motor drivers to arm control, from embedded to Linux, from real-time control to simulation — we cover it all:

- **Motor Drivers** — Plug in CAN bus and go. Damiao, RobStride, MyActuator, HighTorque, Hexfellow — one API for all.
- **Embedded & MCU** — Direct motor control from microcontrollers (ESP32 and more).
- **Cross-Platform** — Linux, macOS, Windows. x86 and ARM. Rust for performance, Python for ease, C/C++ ABI for ecosystem integration.
- **Simulation & Algorithms** — FK/IK, trajectory planning (MoveJ/MoveL/MoveC), offline verification.
- **Robotic Arm SDK** — Full stack: connect, enable, move, calibrate, monitor — out of the box.
- **Debugging & Tooling** — Web dashboard, CLI, parameter tuning, calibration workflows.
